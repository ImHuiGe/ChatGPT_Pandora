## **环境变量**
#### `CHATGPT_API_PREFIX`  
对话 api 请求地址，默认为 `https://ai.fakeopen.com`，可以设置为你的部署后的域名（仅限 vercel），如默认网址 `https://xxxxx.vercel.app`，或其他反向代理地址（任意平台）。

#### `LOGIN_LOCAL`  
是否启用账号密码登录。Pandora 现已支持直接登录，故默认为 `True`或 1。
